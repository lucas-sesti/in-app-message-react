# In App Message

It's a simple example of real time message app

## Autores

- [@lucassesti](https://www.github.com/lucas-sesti)

## Instalação

Install in-app-message with npm or yarn

```bash
  cd in-app-message
  npm install (or yarn)
  npm start (or yarn start)
```
